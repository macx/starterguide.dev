## Neue Website erstellen

Du hast deine neue Website ruckzuck erstellt. Richte dein Projekt direkt über das [Terminal](/guides/terminal) deines Betriebssystems ein oder händisch über Visual Studio Code.

<Tabs>
  <TabItem label="Terminal">
    1. Öffne das Terminal auf deinem Computer.

        Das Terminal startet in der Regel in deinem Benutzerverzeichnis `~/`. Wenn du das Projekt an einem anderen Ort installieren möchtest, navigiere mit `cd` dorthin. Wie das funktioniert, erfährst du im Hilfstext [Aktuellen Pfad kopieren](/guides/terminal#aktuellen-pfad-kopieren).

    2. Führe im Terminal den folgenden Befehl aus, um deine Website zu erstellen:

        ```shell
        npm create website-starterkit
        ```

    3. Folge den Anweisungen, um das Projekt einzurichten.

  </TabItem>

  <TabItem label="Visual Studio Code">
    <Steps>

    1. Erstelle dir auf deiner Festplatte einen neuen Ordner für dein Projekt.

    2. Öffne Visual Studio Code und wähle den neuen Ordner über `Datei` > `Ordner öffnen`.

    3. Zeige nun das Terminal in Visual Studio Code über das Menü `Anzeige` > `Terminal` oder `Terminal` > `Neues Terminal` an.

    4. Führe im Terminal den folgenden Befehl aus, um deine Website zu erstellen:

        ```shell
        npm create website-starterkit
        ```

    5. Folge den Anweisungen, um das Projekt einzurichten.

        :::note
        Wenn du gefragt wirst, wo das Projekt gespeichert werden soll, wähle die Option **Im aktuellen Verzeichnis**.
        :::

    </Steps>

  </TabItem>
</Tabs>

## Bestehende Website bearbeiten

Nachdem du die Website eingerichtet hast, reicht es, deinen Ordner in Visual Studio Code zu öffnen folgenden Befehl ins Terminal einzugeben, um den Entwicklungsserver zu starten:

```shell
npm run dev
```
