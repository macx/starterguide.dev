## Neue Website erstellen

Deine neue Website ist mit dem Starterkit ruck zuck erstellt. Du kannst dein Projekt entweder direkt über die Kommandozeile deines Betriebssystems einrichten, oder händisch über Visual Studio Code.

<Tabs>
  <TabItem label="Kommandozeile">
    1. Öffne das Terminal auf deinem Computer.

    2. Das Terminal startet in der Regel im Benutzerverzeichnis `~/`. Wenn du das Projekt an einem anderen Ort installieren möchtest, navigiere dorthin.

        ```shell
        cd ~/Pfad/zum/Verzeichnis
        ```

    3. Führe im Terminal den folgenden Befehl aus, um deine Website zu erstellen:

        ```shell
        npm create website-starterkit
        ```

    4. Folge den Anweisungen, um das Projekt einzurichten.

    5. Öffne anschließend das Projekt in Visual Studio Code und starte den Entwicklungsserver:

        ```shell
        cd .
        npm run dev
        ```

        Deine Website wird nun im Browser angezeigt. Änderungen, die du an deiner Website vornimmst, werden automatisch im Browser aktualisiert.

  </TabItem>

  <TabItem label="Visual Studio Code">
    <Steps>

    1. Erstelle dir auf deiner Festplatte einen neuen Ordner für dein Projekt.

    2. Öffne Visual Studio Code und öffne über das Menü über `Datei` > `Ordner öffnen` den neuen Ordner.

    3. Öffne das Terminal in Visual Studio Code über das Menü `Anzeige` > `Terminal` oder `Terminal` > `Neues Terminal`.

    4. Führe im Terminal den folgenden Befehl aus, um deine Website zu erstellen:

        ```shell
        npm create website-starterkit
        ```

    5. Folge den Anweisungen, um das Projekt einzurichten.

        :::note
        Wenn du gefragt wirst, wo das Projekt gespeichert werden soll, wähle die Option **Im aktuellen Verzeichnis**.
        :::

    6. Starte nun den Entwicklungsserver:

        ```shell
        npm run dev
        ```

        Deine Website wird nun im Browser angezeigt. Änderungen, die du an deiner Website vornimmst, werden automatisch im Browser aktualisiert.

    </Steps>

  </TabItem>
</Tabs>

## Bestehende Website bearbeiten

Nachdem du die Website eingerichtet hast, reicht es, deinen Ordner in Visual Studio Code zu öffnen folgenden Befehl ins Terminal einzugeben, um den Entwicklungsserver zu starten:

```shell
npm run dev
```
